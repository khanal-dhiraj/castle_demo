# Insta-Engineer

A self-working software engineer. You write the issue; an agent writes the code.

Insta-Engineer is two things wearing the same name:

1. A state machine that drives GitHub issues from "ready" to "shipped" without a human in the implementation loop.
2. The static site you're reading right now, built by that very state machine — page by page, plugin by plugin.

Every page on this site was rendered by a plugin that an agent wrote in its own pull request. By the time you're reading this, the agents that built this site have explained the system that built them.
